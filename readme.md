# Development mode in app

Click the Kaboom logo 7 times to enter development mode.


# Installation  
  
Made using Preact  
  
1. Install NodeJS + NPM + Vite  
2. `npm i`  
3. Create `.env` variable: `VITE_PEERJS_ID=<random-alphanumeric-text>`  
4. `npm run dev` or `npm run build`
